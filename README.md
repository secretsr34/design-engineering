# Sabah Rahman - Design Engineering

Selected front-end work. Everything here is hand-built HTML, CSS, and vanilla JavaScript - no frameworks, no build step. Each folder is a self-contained project with its own README covering the problem, the design decisions, and the implementation.

I design enterprise products, engineer them in production code, and hold a pending US patent in AI explainability. The patent work itself is not in this repository; what is here is the craft around it.

## Projects

| Project | What it is |
|---|---|
| [`decision-intelligence-ui/`](decision-intelligence-ui/) | The interface layer of my Decision Intelligence system - decision registry, integrity breakdowns, counterfactuals, and temporal projections, rebuilt on fixture data. |
| [`f1-pitwall/`](f1-pitwall/) | Pit Wall - an interactive race engineer's decision surface. Strategy, timing, and telemetry presented the way a pit wall actually consumes it. |
| [`portfolio-site/`](portfolio-site/) | The production source of [sabahrahman.com](https://www.sabahrahman.com) - a fast, dependency-free portfolio with case studies. |

## How to run anything here

Every project is static. From a project folder:

```
python3 -m http.server 8000
```

Then open `http://localhost:8000`. No install, no build.
